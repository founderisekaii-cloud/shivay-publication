# **App Name**: Shivay Portal

## Core Features:

- Home Page Showcase: Visually appealing hero section with a publication tagline and animated banner to capture user attention immediately.
- Dynamic PDF Viewer: An integrated PDF viewer with Firebase Storage fetching to view publications with the ability to download. This is linked to dynamic updates and uses sample publications when real ones aren't available.
- Customer Review Summarizer: Generative AI tool to automatically summarize and highlight key points from customer reviews. This will help admins quickly understand feedback.
- Razorpay Payment Mockup: Integration of Razorpay for payments with a demo mode. The module remains inactive until real API keys are added.
- Staff Profile Grid: Staff portfolio section, displaying staff images, names, roles and publications. Display sample data if Firestore is not connected.
- Admin Announcement Feed: Administrators can post announcements and updates to be displayed to users of the website.
- Secure Submission Upload: Users can submit their articles/manuscripts to be uploaded and stored in Firebase Storage for consideration. Shows upload progress and simulates success if the backend is not active.

## Style Guidelines:

- Primary color: Deep crimson (#8B0000), reflecting the gravitas and history associated with publications.
- Background color: Light tan (#F5F5DC), offering a neutral and soft canvas that mimics paper.
- Accent color: Golden yellow (#FFD700), inspired by the 'golden age of print', adding highlights.
- Body text: 'Literata' serif font for readability, contributing to the literary feel.
- Headline Font: 'Playfair', for an elegant, high-end feel
- Elegant, professional icons representing pages and actions.
- Subtle transitions, hover effects, and fade-ins using Framer Motion for fluid user experience.
